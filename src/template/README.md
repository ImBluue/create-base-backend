# <%= name %>

## Setup

1. Go to project folder

> cd <%= name %>

2. Install Dependencies

> npm install

3. Copy .env.template to .env and fill it

## Run

1. Start Server

> npm run start

2. Open in url or using an Endpoint tester (postman)

> http://127.0.0.1:3000/

## Lint (ESLint)

- Check

> npm run lint:check

- Fix

> npm run lint:fix

## Format (Prettier)

- Check

> npm run format:check

- Write

> npm run format:write
